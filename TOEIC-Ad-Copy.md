## 2. Facebook & Instagram Ads

**Hook (0-3 วินาที):**
> "สอน TOEIC มา 25 ปี สิ่งที่ทำให้ส่วนใหญ่สอบไม่ผ่านคือ..."

**Body (3-12 วินาที):**
> "ไม่ใช่เรื่องคำศัพท์ แต่คือ 'วิธีฟัง' ที่ผิด

> ผมสอนเทคนิคนี้ในคลาสแรกเลย"

## 3. TikTok Ads

### 3.1 InFeed Ad — "Hook Formula" (ตาม TikTok Algorithm 2026)

**Format: Hook → Problem → Solution → Proof → CTA**

**Version A: Shock Hook**
```
[0-1 วิ]  ภาพ: คะแนน TOEIC 990 บนหน้าจอ
Text:     "นักเรียนผมสอบได้ 990 ใน 3 เดือน — นี่คือที่เขาใช้เวลาส่วนใหญ่ไป"
[1-5 วิ]  ขึ้น List 3 ข้อ (ไม่ใช่สิ่งที่คิด)
[5-12 วิ] อธิบาย Technique จริง 1 ข้อ
[12-15 วิ] "ติดตามเพื่อดูเทคนิคที่เหลือ + ทดลองเรียนฟรีในลิงก์"
```

**Version B: Relatability Hook**
```
[0-2 วิ]  "POV: คุณสอบ TOEIC มาแล้ว 3 ครั้ง แต่คะแนนไม่ขึ้นสักที"
[2-8 วิ]  Pain points จริง ๆ ที่คนเจอ (แบบ empathetic)
[8-13 วิ]  "ปัญหาไม่ได้อยู่ที่ความสามารถ — อยู่ที่ Method"
[13-15 วิ] CTA: "ให้ผมช่วยวิเคราะห์ให้ฟรี — กด Bio Link"
```
